🔧 Project Name

R6 logitech recoil script (Beta)

A customizable Logitech script designed to reduce recoil patterns for supported games. Currently in beta — expect bugs and ongoing changes.

⚠️ Disclaimer
This is a beta release and may be unstable
Use at your own risk
May not work perfectly across all setups (DPI, sensitivity, in-game settings) so read the script and the exact settings will be there
Undetected in r6 as of (April 2026)
✨ Features
Adjustable recoil control
Weapon/profile switching
Sensitivity/DPI scaling support
Toggle on/off keybind
Lightweight (runs via Logitech software)
🧪 Beta Status
Some weapons/patterns still being tuned
Occasional jitter or overcompensation
Limited testing across devices/games
🖥️ Requirements
Logitech mouse compatible with G Hub or LGS
Correct DPI & in-game sensitivity setup
⚙️ Installation
Open Logitech G Hub / LGS
Create a new script/profile
Paste the script code
Assign toggle key
Save and test in-game
🎮 Configuration
DPI values
Sensitivity multipliers
Weapon profiles
Keybinds
-- Example config
DPI = 800
SENS = 6,6
🚧 Known Issues
Doesn’t scale perfectly at extreme sensitivities
Some weapons may feel inconsistent
May conflict with certain G Hub versions
